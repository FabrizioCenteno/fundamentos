Ejercicio 12

Crea una función que tome un array de números y devuelva un nuevo array con solo los números pares.Usa filter() y el operador % .

const arr = [1,2,3,4,5,6,7,8,9];

function pares(arr){

return arr.filter( element => element % 2 == 0)

}

const arrayPares = pares(arr);

console.log(arrayPares);
