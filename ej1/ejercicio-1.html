Fundamentos: Ejercicios JavaScript

Ejercicio 1

Declara variables de varios tipos. Muestra el contenido en la consola junto con el tipo de dato. En este ejercicio utiliza el console.log, console.error, console.info y el console.debug.

const a ='a', b = 'b', c='c', d='d';

console.log(a);

console.error(b);

console.info(c);

console.debug(a);
