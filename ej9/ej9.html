Ejercicio 9

Escribe una función que tome una cadena como argumento y devuelva la cadena invertida. Usa el método split() para convertir la cadena en un arreglo, reverse() para invertir el arreglo, y join() para unirlo de nuevo.

const arr = prompt('ingresa la palabra pa darle vuelta');

function revertirCadena(cadena) {

return cadena.split('').reverse().join('');

}
