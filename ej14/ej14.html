Ejercicio 14

Verificar si una palabra es un palíndromo Escribe una función que verifique si una palabra es un palíndromo (es decir, si se lee igual al derecho y al revés).

const palabra = prompt('ingresa una palabra');

let laVuelta = '';

function comprobar(palabra){

for( let i = palabra.length - 1; i >= 0; i--){

laVuelta += palabra[i];

}

console.log(`${palabra} == ${laVuelta} ? ${palabra == laVuelta ? 'son iguales' : 'no son iguales'}`);

}

comprobar(palabra);
