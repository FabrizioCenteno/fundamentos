Ejercicio 8

Escribe una función que permita generar un número aleatorio entero entre un intervalo. Por ejemplo azar(-5, 10). Salida: 3

const min = prompt('ingresa el valor mínimo');

const max = prompt('ingresa el valor máximo');

function getRandom(min,max){

return Math.floor(Math.random() * (Number(max) - Number(min) + 1)) + Number(min);

}

console.log(getRandom(min,max));
