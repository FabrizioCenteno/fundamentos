Ejercicio 10

Crea una función que encuentre el número más grande en un array. Usa el método Math.max() y el operador ... para "expandir" el array

const arr = [1,2,3,4,5,6,7];

let max = arr[0];

for(let i = 1; i < arr.length; i++){

if(arr[i] > max) max=arr[i];

}

console.log(max);
